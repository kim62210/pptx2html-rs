This page tracks the shipped 1.0.5 release line and demonstrates the current stable in-browser PPTX → HTML conversion flow. Drop a PowerPoint file below to inspect slide count, canvas size, and rendered HTML output without leaving the browser.

Whole-slide image-like zoom controls are being prepared separately for the next 1.1.0 line. This stable demo intentionally stays aligned with what was actually released in 1.0.5.
